## üÜò TROUBLESHOOTING

### "Calendar not loading"
- Check URL is correct in HTML
- Make sure GHL calendar is published
- Try different browser
- Clear cache

### "Buttons don't work"
- Check links in HTML match your GHL URL
- Disable pop-up blocker
- Check browser console for errors

### "Can't select times"
- Set availability in GHL dashboard
- Make sure calendar is set to "Public"
- Check time zone settings

### "Bookings not showing in GHL"
- Verify calendar URL is correct
- Check GHL calendar settings
- Contact GHL support
